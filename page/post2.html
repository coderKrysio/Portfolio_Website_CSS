Slider Game

About

Developed by using simple HTML tag elements, this game has some basic functionality. The game is played by arranging the slides on top of each other to each the 25th slide.

Disclaimer: This project development is an adaptive idea. I do not own any of the imported images or the sound from the online resources. All credits to its rightful owners.

Purpose

To learn the usage of the HTML tag elements with some basic CSS and Javascript.

Problems Faced

While adding graphics to the background, the slider limit needed to be changed and I solved it by using a variable that calculates values based on compatibility.

Original Idea

The original idea belongs to KnifeCircus from youtube.
Knife Circus: Javascript Slider Game Tutorial

Modifications Made

By adding scoreboard, instruction and background graphics, the gameboard was made responsive and compatible as well.
